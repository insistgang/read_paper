# 推荐插件列表

## 📦 核心插件（必装）

### 1. Templater
- **用途**: 高级模板系统
- **设置**:
  - Template folder location: `03-Templates`
  - Trigger Templater on new file creation: ✅
  - Enable system command: ✅

### 2. PDF++ (原 Annotator)
- **用途**: PDF阅读和标注
- **功能**:
  - 在Obsidian内直接打开PDF
  - 高亮、下划线、批注
  - 标注自动同步到笔记

### 3. Zotero Integration
- **用途**: Zotero文献管理集成
- **功能**:
  - 自动导入Zotero文献
  - 生成论文笔记
  - 同步PDF和元数据

### 4. Dataview
- **用途**: 数据查询和展示
- **示例查询**:
  ```dataview
  TABLE year, venue, rating
  FROM #paper
  SORT date_added DESC
  ```

### 5. Tag Wrangler
- **用途**: 标签管理
- **功能**:
  - 批量重命名标签
  - 标签层级管理

## 🤖 AI相关插件

### 6. Smart Connections
- **用途**: AI驱动的笔记关联
- **功能**:
  - 自动发现相关笔记
  - 语义搜索
  - AI对话

### 7. Copilot
- **用途**: AI助手
- **功能**:
  - 笔记总结
  - 问答对话
  - 内容生成

### 8. Local GPT
- **用途**: 本地AI处理
- **功能**:
  - 使用本地LLM
  - 保护隐私
  - 离线可用

## 📝 笔记增强插件

### 9. Admonition
- **用途**: 美化提示框
- **用法**:
  ```ad-note
  title: 重要发现
  内容...
  ```

### 10. Mermaid Tools
- **用途**: 图表绘制
- **功能**:
  - 流程图
  - 时序图
  - 思维导图

### 11. Kanban
- **用途**: 看板视图
- **用途**:
  - 论文阅读计划
  - 项目进度管理

### 12. Calendar
- **用途**: 日历视图
- **功能**:
  - 每日笔记导航
  - 阅读记录可视化

## 🔧 工具类插件

### 13. Obsidian Git
- **用途**: 版本控制
- **功能**:
  - 自动备份
  - 多设备同步

### 14. Advanced Tables
- **用途**: 表格增强
- **功能**:
  - 快捷键操作
  - 公式计算

### 15. Excalidraw
- **用途**: 手绘图
- **功能**:
  - 概念图绘制
  - 架构图设计

### 16. Highlightr
- **用途**: 文本高亮
- **功能**:
  - 多种颜色高亮
  - 快捷键操作

## 📊 数据可视化插件

### 17. DB Folder
- **用途**: 数据库视图
- **功能**:
  - 类似Notion的数据库
  - 多视图切换

### 18. Charts
- **用途**: 图表生成
- **功能**:
  - 柱状图、折线图、饼图
  - 数据可视化

## 🎨 外观插件

### 19. Minimal Theme
- **用途**: 简洁主题
- **特点**: 现代、干净

### 20. Style Settings
- **用途**: 样式自定义
- **功能**: 细粒度样式控制

## ⚙️ 安装步骤

1. 打开 Obsidian 设置
2. 进入 "第三方插件" → "浏览"
3. 搜索并安装上述插件
4. 启用插件并配置

## 📋 推荐安装顺序

1. **第一批**（立即安装）:
   - Templater
   - PDF++ / Annotator
   - Dataview
   - Calendar

2. **第二批**（一周内）:
   - Zotero Integration
   - Tag Wrangler
   - Obsidian Git
   - Admonition

3. **第三批**（按需安装）:
   - AI相关插件
   - 可视化插件
   - 外观插件
